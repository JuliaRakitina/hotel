Настоящая папка содержит исходные коды демонстрационного плагина для iikoOffice.
В исходных кодах проекта приводятся примеры использования API iikoOffice, предназначенного для разработчиков сторонних плагинов.
Сборка Resto.BackApi.Core.dll вместе с документирующим XML-файлом находится в подпапке lib.
